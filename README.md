# projektlabor_festeknyilvantarto

**CONSOLE COMMANDS:**
 - `nvm install 20`
 - `nvm use 20`
 - `npm i -g yarn`
 - `yarn`
 - `vite`


**Run App**
- `yarn dev`

**Run Error because Yarn Version**
- `corepack enable`
- `corepack prepare yarn@4.4.1 --activate`

**Axios Download**
- `npm install axios`
- `npm install --save-dev @types/axios`
